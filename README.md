# Expense Tracker

A Java-based Expense Tracker application using JDBC and MySQL.

## Features
- Add Expense
- View Expenses
- Delete Expense
- Search by Category
- Search by Date
- Monthly Report
- MySQL Database Integration
- Input Validation

## Technologies Used
- Java
- JDBC
- MySQL
- Git & GitHub

## Project Structure
- DBConnection.java
- Expense.java
- ExpenseDAO.java
- Main.java

## How to Run
1. Clone repository
2. Configure MySQL database
3. Update DB credentials
4. Run Main.java
